Aula Git e GitHub

Atividade

1) • Realize o cadastro no Github;
• Siga o perfil do professor Picolo https://github.com/luizpicolo;
• Crie um repositorio do GitHub com seu nome de usuario mais github.io. Por exemplo, se seu usuario for alunoifms, crie um repositorio com o nome alunoifms.github.io;
• Importe o repositorio criado para o Replit;
• No projeto importado, crie um arquivo index.html e adicione codigo HTML para uma pagina pessoal, voce decide o que devera conter sua pagina;
• Adicione as modificacoes usando os comandos git aprendidos em aula;
– git add index.html;
– git commit -m ”Mensagem do que foi feito”;
• Envie os arquivos para o GitHub;
– git push origin master;
• Acesse o GIT e verifique se os arquivos foram adicionados corretamente e acesse a URL do seu novo site, por exemplo, alunoifms.github.io, onde alunoifms deve ser alterado para o seu usuario.
